이곳에서 모스봇에 대한 모든 정보를 찾을 수 있습니다.

  • 봇 정보: 모스봇#1008
  • 봇 개발자: Boss#3333
  • 봇 운영 시작일: 2020-07-17
  • 봇 마지막 업데이트: 2022-07-11

궁금한 점이 있다면 서포트 서버에서 질문해 주시거나 Boss#3333으로 연락 부탁드립니다.

초대하기

아래 방법 중 하나를 따라해 모스봇을 초대할 수 있습니다.

  1. 디스코드 내에서 모스봇의 프로필을 눌러 [서버에 추가]버튼을 눌러 본인의 디스코드 서버로 초대할 수 있습니다.
  2. 링크를 눌러 본인의 디스코드 서버로 초대할 수 있습니다.

모스봇을 이용하기 전에 개인정보처리방침을 읽어 주세요.

데이터베이스 만들기

모스봇을 서버에서 이용하기 위해서는 데이터베이스가 필요합니다.

데이터베이스에는 기본 데이터베이스, 후기 데이터베이스가 존재합니다.

기본 데이터베이스에는 모스봇을 이용한 유저의 데이터(포인트, 출석 정보 등),
모스봇의 게임 데이터(꽝의 개수, 게임 활성화 여부 등), 경고 데이터(경고자, 경고수, 경고이유)들을
저장하기 위해 필요한 데이터베이스 입니다.

기본 데이터베이스의 구조는 이러합니다.


[기본 데이터베이스] ＜── 데이터베이스만 만들면 하위 데이터베이스들은 자동 생성됩니다.
       │
       ├─[유저 데이터베이스]
       │       ├─(유저 데이터)
       │       │     └─포인트 정보, 출석 정보들
       │       ├─(유저 데이터)
       │       │     └─......
       │       └─......
       │             └─......
       │
       ├─[서버 설정 데이터베이스]
       │       ├─(게임 설정 데이터)
       │       └─(기타 데이터)
       │
       └─[경고 데이터베이스]
               ├─(유저 데이터)
               │      └─경고 정보들
               ├─(유저 데이터)
               │      └─......
               └─......
                      └─......

기본 데이터베이스는 서버 당 하나만 필요하며
기본 데이터베이스를 만들기 위해서는 아래 명령어를 입력해야 합니다.

/데이터관리 서버데이터관리 종류:기본 데이터베이스 옵션:데이터 베이스 설정하기


옵션에 대한 정보는 데이터 관리 명령어 카테고리에서 확인하실 수 있습니다.

후기를 올렸을 때의 정보(후기 대상, 작성자, 개수)를 저장하기 위해 필요한 데이터베이스 입니다.

      ├─(유저 데이터)

            └─......

공지 이름 규칙

  • 한글, 영어, 숫자로만 이름을 지을 수 있습니다.

  • 25개까지만 만들 수 있습니다.
  • 이미 있는 공지의 이름은 사용할 수 없습니다.

게임 명령어

모스봇의 게임 명령어입니다.

  • /게임 기억력 - 기억력 게임을 완료하고 포인트를 받습니다.
  • /게임 내포인트 (유저) - 본인 혹은 다른 유저의 포인트를 확인합니다.
  • /게임 뽑기 [배팅포인트)] - 배팅한 포인트를 걸고 뽑기 게임을 진행합니다.
  • /게임 슬롯머신 [배팅포인트] - 배팅한 포인트를 걸고 주사위 게임을 진행합니다.
  • /게임 주사위 [배팅포인트] - 배팅한 포인트를 걸고 슬롯머신 게임을 진행합니다.
  • /게임 출석 - 출석체크하고 포인트를 받습니다.
  • /게임 코인거래 [거래방식] - 가상 코인을 매수하거나 매도합니다.
  • /게임 코인정보 [옵션] - 가상 코인의 정보를 확인합니다.
  • /게임 포인트랭킹 - 명령어를 입력한 서버에서의 포인트 랭킹을 확인합니다.
  • /게임 포인트전송 [유저] [포인트] - 유저에게 본인의 포인트를 전송합니다.

대부분의 게임 명령어는 기본 데이터베이스가 있어야 합니다.

아이템은 1.5배, 2.0배의 아이템이 있으며 각각 배팅 포인트의 1.5배, 2.0배를 추가로 얻으실 수 있습니다.
