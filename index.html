    모스봇이 무엇인가요?

    모스봇은 2020년 6월에 파이브엠 커뮤니티 서버인
    [My own script] 나만의 스크립트 서버에서 운영진의 서버 관리를 돕고
    유저들의 재미를 위한 다양한 게임들을 제공하기 위해 만들어진 봇입니다.

    2021년 8월부터 모스봇을 공개 봇으로 전환하며 여러 서버에 관리, 게임 및 각종 유틸리티 기능들을 제공하고 있습니다.

    필요한 권한들..

    필수 권한

    '링크 첨부' = 모스봇은 임베드 메시지를 사용하기에 필수적으로 필요합니다.
    '메시지 기록 보기' = 명령어를 입력받고 이벤트를 발생시키기 위해 필요합니다.
    '메시지 보내기' = 명령어에 반응하기 위해서 필요합니다.

    선택 권한

    '메시지 관리' = 모스야 청소 명령어를 사용하려면 필요합니다.
    '멤버 추방하기' = 모스야 추방해 명령어를 사용하려면 필요합니다.
    '멤버 차단하기' = 모스야 차단해 명령어를 사용하려면 필요합니다.
    '서버 관리하기' = 모스야 서버정보 명령어를 사용하려면 필요합니다.

    명령어

    아이콘 안내


    이 아이콘이 붙은 명령어는 디스코드 서버마다
    모스야 서버 데이터 만들어를 입력해 서버 데이터를 생성해야 사용할 수 있는 명령어입니다.


    이 아이콘이 붙은 명령어는 디스코드 서버마다
    모스야 후기 데이터 만들어를 입력해 후기 데이터를 생성해야 사용할 수 있는 명령어입니다.


    이 아이콘이 붙은 명령어는 서버 주인만 사용할 수 있는 명령어입니다.


    이 아이콘이 붙은 명령어는 특정 권한이 있어야 사용할 수 있는 명령어입니다.

    Tip.

    커뮤니티 명령어

    모스야 출근해 = 입력한 채널에 출근 임베드 메시지를 출력합니다. (출근 시 퇴근 버튼이 생성됨)
    모스야 경고해 (경고할 유저의 아이디 혹은 멘션) (사유) = 유저에게 경고를 추가합니다.
    모스야 경고지워 (경고 지울 유저의 아이디 혹은 멘션) (지울 경고 개수) = 유저에게 경고를 개수만큼 지웁니다.
    모스야 경고확인 (경고 확인할 유저의 아이디 혹은 멘션) = 유저에게 몇 개의 경고가 있는지 확인합니다.

    유틸리티 명령어

    모스야 청소 (개수) = 개수만큼 메시지를 삭제합니다.
    모스야 차단해 (차단할 유저의 아이디) (사유) = 유저를 차단합니다. (ban)
    모스야 추방해 (추방할 유저의 아이디) (사유) = 유저를 추방합니다. (kick)
    모스야 로그채널 설정해 = 차단, 추방, 경고 명령어 이용 시 로그를 올릴 채널을 선택합니다.
    모스야 내정보 = 본인의 디스코드 정보를 확인합니다.
    모스야 서버정보 = 현재 서버의 정보를 확인합니다.
    모스야 도움말 = 모스봇의 모든 명령어를 확인합니다.

    코인게임 명령어 (가상)

    모스야 코인 정보 = 현재 코인의 가격과 가격 그래프, 본인의 누적 수익률 등을 확인합니다.
    모스야 코인 매수 = 코인을 매수(구매)합니다.
    모스야 코인 매도 = 코인을 매도(판매)합니다.
    모스야 코인 그래프 = 최근 1시간 동안 변동된 코인의 가격을 그래프로 확인합니다.
    모스야 코인 태그 변경 = 코인 순위에 등록된 본인의 고유 태그를 변경합니다.
    모스야 코인 순위 = 코인 수익률을 순위로 매겨 확인합니다. (모든 유저가 하나의 순위표에 보여집니다.)
    모스야 코인 기록 삭제 = 본인의 코인 수익률, 태그 등을 초기화합니다.

    후기 명령어

    모스야 후기 데이터 만들어 = 현재 서버의 후기 데이터베이스를 생성하여 후기 시스템을 이용할 수 있게 만듭니다.
    모스야 후기올려 (후기 올릴 상대 유저의 아이디) (후기 내용) = 상대 유저에게 후기를 등록합니다.
    모스야 후기랭킹 = 현재 서버에서 후기를 많이 받은 유저 순서대로 순위를 매겨 확입니다.
    모스야 후기조회 = 본인에게 등록된 후기 개수를 확인합니다.

    게임 명령어

    모스야 출석 = 출석체크를 하고 포인트를 받습니다.
    모스야 주사위 (베팅할 포인트) = 베팅한 포인트로 주사위 게임을 시작합니다.
    모스야 타자 (영어 혹은 한글) = 영어 혹은 한글 문장으로 타자게임을 시작합니다.
    모스야 뽑기 (베팅할 포인트) = 베팅한 포인트로 뽑기 게임을 시작합니다.
    모스야 버그해결 뽑기 = 뽑기 게임이 정상적으로 작동하지 않을 때 입력하면 고칠 수 있습니다.
    모스야 버그해결 주사위 = 주사위 게임이 정상적으로 작동하지 않을 때 입력하면 고칠 수 있습니다.

    게임 및 서버 데이터 관리 명령어

    모스야 서버 데이터 만들어 = 현재 서버의 데이터베이스를 생성하여 게임 및 유틸 기능의 속성을 설정할 수 있게 만듭니다.
    모스야 서버 데이터 지워 = 현재 서버의 데이터베이스를 삭제합니다. (모든 게임 및 유틸 기능, 유저 데이터들이 삭제됩니다.)
    모스야 서버 데이터 설정 = 현재 서버의 게임 기능, 유틸 기능들의 속성값들을 입맛대로 수정합니다.
    모스야 서버 데이터 확인해줘 = 현재 서버의 게임 기능, 유틸 기능들의 속성값들을 확인합니다.
    모스야 유저 데이터 지워 (유저 아이디) = 선택한 유저의 데이터를 삭제합니다. (저장된 포인트 등)
    모스야 포인트 지급해 (포인트 받을 유저의 아이디) = 선택한 유저에게 포인트를 지급합니다.
    모스야 포인트 차감해 (포인트가 차감될 유저의 아이디) = 선택한 유저의 포인트를 차감합니다.
    모스야 내 데이터 만들어 = 본인의 데이터베이스를 서버 데이터베이스에 추가합니다. (포인트 기록 용도)
    모스야 내 데이터 지워 = 본인의 데이터베이스를 서버 데이터베이스에서 삭제합니다. (포인트가 초기화 됩니다.)
    모스야 내 데이터 조회해 = 현재 서버에서의 본인 포인트를 확인합니다.
    모스야 포인트 랭킹 = 현재 서버에서의 포인트 순위를 확인합니다.
    모스야 포인트 보내 (포인트를 받을 유저의 아이디) = 상대 유저에게 자신의 포인트를 나눠줍니다. (보낼 포인트 양 조절 가능)

    파이브엠 게임 데이터베이스 관리 명령어

    모스야 파엠 데이터베이스설정 = 모스봇을 본인의 파이브엠 게임 서버 데이터베이스에 접속할 수 있도록 설정합니다.
    모스야 파엠 명령어설정 = 파엠 관련 명령어마다 사용할 수 있는 역할을 지정합니다.
    모스야 파엠 데베정보삭제 = 파엠데이터베이스 접속 설정값 및 파엠명령어 설정값들을 삭제합니다.
    모스야 파엠 데베접속확인 = 모스봇이 정상적으로 본인의 파이브엠 서버의 데이터베이스에 접근할 수 있는지 확인합니다.
    모스야 파엠 고유번호확인 (확인할 고유번호) = 해당 고유번호를 가진 유저의 게임 데이터를 확인합니다.
    모스야 파엠 내고유번호확인 = 본인의 게임 속 고유번호를 확인할 수 있습니다.
    모스야 파엠 밴추가 (밴할 고유번호) = 선택한 고유번호를 밴합니다.
    모스야 파엠 밴제거 (언밴할 고유번호) = 선택한 고유번호의 밴을 해제합니다.
    모스야 파엠 화이트리스트추가 (화이트리스트에 추가할 고유번호) = 선택한 고유번호를 화이트리스트에 추가합니다.
    모스야 파엠 화이트리스트제거 (화이트리스트에서 제거할 고유번호) = 선택한 고유번호를 화이트리스트에서 제거합니다.
    모스야 파엠 지갑추가 (돈을 지급할 고유번호) (금액) = 선택한 고유번호의 지갑에 금액만큼 돈을 추가합니다.
    모스야 파엠 지갑제거 (돈을 차감할 고유번호) (금액) = 선택한 고유번호의 지갑에서 금액만큼 돈을 차감합니다.
    모스야 파엠 계좌추가 (돈을 지급할 고유번호) (금액) = 선택한 고유번호의 계좌에 금액만큼 돈을 추가합니다.
    모스야 파엠 계좌제거 (돈을 차감할 고유번호) (금액) = 선택한 고유번호의 계좌에서 금액만큼 돈을 차감합니다.
    모스야 파엠 크레딧추가 (크레딧을 지급할 고유번호) (액수) = 선택한 고유번호에게 액수만큼 크레딧을 지급합니다.
    모스야 파엠 크레딧제거 (크레딧을 차감할 고유번호) (액수) = 선택한 고유번호의 크레딧을 액수만큼 차감합니다.
    모스야 파엠 차량추가 (차량을 지급할 고유번호) (차량 코드) = 선택한 고유번호의 차량 목록에 차량 코드를 추가합니다.
    모스야 파엠 차량제거 (차량을 제거할 고유번호) (차량 코드) = 선택한 고유번호의 차량 목록에서 차량 코드를 삭제합니다.
    모스야 파엠 그룹조회 (조회할 고유번호) = 선택한 고유번호의 그룹 목록을 확인합니다.
    모스야 파엠 그룹추가 (그룹을 추가할 고유번호) (그룹 이름) = 선택한 고유번호의 그룹 목록에 그룹을 추가합니다.
    모스야 파엠 그룹제거 (그룹을 제거할 고유번호) (그룹 이름) = 선택한 고유번호의 그룹 목록에서 그룹을 삭제합니다.
