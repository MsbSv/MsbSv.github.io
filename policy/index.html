서비스 이용약관 (Terms of Service)

제1조 (목적)

본 약관은 모스봇(이하 "서비스")이 제공하는 봇 및 웹 관리패널 서비스(이하 "서비스")의 이용과 관련하여 회사와 이용자 간의 권리·의무·책임사항 및 기타 필요한 사항을 규정함을 목적으로 합니다.

제2조 (정의)

  • 서비스: Discord 봇 및 웹 기반 관리패널 일체
  • 이용자: 서비스를 이용하는 모든 개인 또는 단체

제3조 (이용계약의 성립)

이용계약은 이용자가 본 약관에 동의하고 서비스의 사용 신청을 완료함으로써 성립합니다.

제4조 (서비스 제공 범위)

  • 봇의 기본 기능, 대시보드(통계, 실시간 현황 등), 티켓 로그 열람 등
  • 관리자 권한이 필요한 기능의 경우 해당 서버의 관리자 권한 소유자에게만 제공됩니다.

제5조 (금지행위)

  • 봇의 정상 기능을 방해하는 행위
  • 부정한 방법으로 타 계정/서버의 데이터를 수집·보관하는 행위

제6조 (면책 및 책임제한)

서비스는 외부 Discord API 정책 변경, 디스코드 자체 장애, 네트워크 문제 등 서비스 제공자의 통제를 벗어난 사유에 대해 책임을 지지 않습니다. 단, 고의·중대한 과실이 입증된 경우를 제외합니다.

제7조 (계약해지 및 데이터 삭제)

이용자가 서비스 이용을 중단하거나 데이터 삭제를 요청할 경우 사유 확인 후 관련 데이터를 삭제합니다. (로그 보존기간 등 내부 정책을 따름)

제8조 (분쟁해결)

서비스 이용과 관련된 분쟁은 대한민국 법을 따르며, 관할 법원은 서비스 사업자의 소재지 관할 법원으로 합니다.

개인정보 처리방침 (Privacy Policy)

업데이트 2025-12-05

1. 수집하는 정보

  • Discord 식별자: 프로필 ID, 프로필 이름, 아바타 이미지 등
  • 서버 식별자: 서버 ID, 서버 이름, 서버 아이콘 등
  • 티켓 로그: 메시지 내용, 첨부파일 URL, 타임스탬프 (티켓 기능 동작 시 수집)
  • 접속 로그: 관리패널 접속 기록 및 실시간 뷰어 수 (통계/운영 목적)

2. 수집 목적

  • 서비스 제공 및 유지보수
  • 티켓 처리 기록 보관 및 지원
  • 서비스 모니터링(실시간 이벤트), 오류 추적
  • 부정행위 탐지 및 보안

3. 보관 기간

티켓 로그: 기본 90일 (요청 시 즉시 삭제 요청 가능). 실시간 이벤트는 원칙적으로 저장하지 않으며, 저장할 경우 명시적으로 고지합니다.

4. 제3자 제공

법적 요청 또는 사용자 동의가 있는 경우에 한해 제3자에게 제공합니다. 그 외에는 제공하지 않습니다.

5. 이용자의 권리

정보 열람·정정·삭제 요청 가능. 삭제 요청 시 내부 정책과 법적 보존 의무를 검토 후 처리합니다.

6. 기술적 안전조치

  • HTTPS 전송, DB 접근 최소 권한, 암호화(필요 시)
  • 관리자 패널 접근은 OAuth2 인증 및 권한 확인
